
# Portfolio Website for Rong Soavnnrot

A sleek, cybersecurity-themed personal portfolio with a dark blue/black color scheme, animated particle network background, and engaging tech-inspired visual effects.

---

## Design Theme
- **Primary Colors**: Deep blue (#0066ff) for accents, black (#0a0a0f) for backgrounds
- **Effects**: Matrix-inspired subtle glow effects, typing animations, glowing borders on hover
- **Typography**: Clean, modern sans-serif fonts with a tech aesthetic
- **Particles**: Animated network of connected dots and lines in the hero section

---
